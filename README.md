# Java Mini Projects

This repository contains two small Java projects developed for practice and learning purposes.

---

## 1. Calculator Application

**Description:**  
A simple console-based Java calculator that can perform basic arithmetic operations like addition, subtraction, multiplication, and division.

**Features:**
- Addition, Subtraction, Multiplication, Division
- Simple user input via console
- Handles basic errors like division by zero

**Folder:** `calculator/src`

---

## 2. Bank Account System

**Description:**  
A simple Java program to simulate basic banking operations such as deposit, withdraw, and checking balance.

**Features:**
- Deposit and Withdraw money
- Display account balance
- Simple console-based interaction

**Folder:** `bank-system/src`

---

## How to Run
1. Open the project folder in your IDE (like Eclipse or IntelliJ).  
2. Compile the `.java` files inside `src` folder.  
3. Run the main class (`Main.java` or equivalent) in console/IDE.
